# H bridge Motor

In this project,I going to Use the Arduino to control 2 DC motors with H-bridge l298, there are many ways to simulate motors with H-bridge, this is what I did in the project.

# Components

1- Arduino Uno R3

2- 2 DC Motors

3- L293D H Bridge

4- Battery
