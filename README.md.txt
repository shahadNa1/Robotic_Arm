The task of installing the robotic arm and the assembly method for assembling the robotic arm
  Follow the steps:
1 - Download any 3D software

2 - Download the robotic arm parts file via

3 - Connect the each part (base, shoulder, elbow, etc.).
4 - Fix the base with screws on the movable base. 
5 - Attach the base servo motor to the waist.
6 - Attach the clutch to the arm with the arm
7 - Then connect the two groups together
